## Simple android accounting app


### 系統架構及功能簡介

 - 此系統包含記錄不同項目的金額花費、刪除金額花費以及計算不同項目所花費的總額，用資料庫儲存在手機中，將每個項目的花費總金額列出來供使用者查詢，並提供簡潔的Android頁面。
